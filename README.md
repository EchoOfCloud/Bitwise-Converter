# Bitwise Converter

一个功能强大的二进制转换器，支持在二进制、ASCII和Unicode之间进行相互转换。

## 功能特点

- 支持二进制与ASCII互转
- 支持二进制与Unicode互转
- 可自定义字节长度（1-20位）
- 直观的用户界面
- 实时转换结果显示
- 响应式设计，适配不同屏幕尺寸

## 使用方法

1. 在输入框中输入要转换的内容
2. 选择源格式和目标格式
3. 选择适当的字节长度
4. 点击转换按钮查看结果
5. 可以使用复制按钮快速复制结果

## 技术实现

- 纯HTML、CSS和JavaScript实现
- 无外部依赖
- 客户端本地计算，无需服务器
